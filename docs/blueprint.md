# **App Name**: Lade Stack Blog

## Core Features:

- Home Page Blog Listing: Display a grid or list of blog post cards with cover image, title, description, author, and date.
- Single Blog Page: Render dynamic blog content with title, author, date, cover image, typography, and spacing.
- About Page: Minimal bio section with image and text layout.
- Contact Page: Simple contact form (name, email, message) logging data to console on submit.
- Admin Placeholder: Placeholder card for admin dashboard with styled (non-functional) login button.
- Dark Mode Toggle: Support dark mode using Shadcn UI.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) for a calm and professional feel.
- Background color: Light gray (#F9F9F9) for a minimal and clean look.
- Accent color: Dark gray (#555555) for text and important elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for both headlines and body text.
- Minimal and consistent icons throughout the website.
- Clean and minimal layout with generous whitespace.
- Subtle animations and transitions for a smooth user experience.
